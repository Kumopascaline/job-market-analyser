# 🔍 Job Market Analyser — Data Science & AI Skills

**Week 3 of my One Project Per Week Resolution**

## 📋 Project Overview
Analysis of 65 real job descriptions across 16 
countries to answer one question:

**"What skills are data science and AI employers 
actually asking for right now?"**

This is original research using a self-collected 
dataset — nobody else has this exact analysis!

---

## 🌍 Dataset
- **65 job descriptions** collected manually
- **16 countries** including African markets
- **African jobs:** Nigeria, South Africa, 
  Morocco, Douala (Cameroon), Kenya
- **Global jobs:** USA, UK, Ireland, Germany, 
  Canada, Malaysia, Indonesia and more

---

## 🔑 Key Findings

| Skill | Total Demand | Africa | Global |
|---|---|---|---|
| **Machine Learning** | 7 | 4 🥇 | 3 |
| **SQL** | 6 | 1 | 5 🥇 |
| **NLP** | 4 | 2 | 2 |
| **Python** | 4 | 1 | 3 |
| **Power BI** | 3 | 1 | 2 |
| **Collaboration** | 2 | 2 | 0 |
| **MLOps** | 2 | 0 | 2 |

---

## 🌍 Africa vs Global Insights

**What African employers want MORE:**
- Machine Learning — 4 mentions vs 3 globally
- NLP — equal demand but growing in Africa
- Collaboration & soft skills — Africa only!

**What Global employers want MORE:**
- SQL — 5 mentions vs 1 in Africa
- MLOps — deployment skills
- Excel — still widely used globally

**Key insight:**
> African employers are leaning heavily into 
> AI and Machine Learning while global employers 
> still prioritise SQL and data fundamentals.
> To work ANYWHERE — you need both!
